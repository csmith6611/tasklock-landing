# TaskLock — Finish First. Surf Second.

A beautiful landing page for TaskLock, a privacy-first, self-hosted task enforcement system for the home.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- **Vite** - Fast development and build
- **React 18** - Component-based UI
- **Custom CSS** - No framework dependencies, fully custom design

## Design Features

- Dark theme with warm amber accents
- DM Serif Display + Manrope typography pairing
- Smooth scroll-triggered animations
- Responsive across all devices
- Geometric grid background pattern
- Accessible and performant
